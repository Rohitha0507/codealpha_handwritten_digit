# 🧠 DigitVision AI – Handwritten Digit Recognition

An intelligent Handwritten Digit Recognition system developed using Deep Learning and Streamlit. This application predicts handwritten digits (0–9) in real time using a Convolutional Neural Network (CNN) trained on the MNIST dataset.

## 📌 Project Overview

DigitVision AI is designed to recognize handwritten digits from uploaded images with high accuracy. It leverages a Convolutional Neural Network (CNN) trained on the MNIST dataset and provides real-time predictions through a modern Streamlit web interface.

---

## ✨ Features

- 🔢 Handwritten Digit Recognition (0–9)
- 🧠 Deep Learning using CNN
- 📷 Upload handwritten digit images
- ⚡ Real-time prediction
- 📊 Confidence score visualization
- 📈 Prediction probability chart
- 🎨 Modern and responsive Streamlit UI
- 🌐 Deployable on Streamlit Cloud

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/codealpha_handwritten_digit.git
cd codealpha_handwritten_digit
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
